Projekt s 'NETFLIX' vytvořený pomocí HTML, CSS a JS dle přiloženého souboru 'Zadání.pdf'.

Pro spuštění projektu zadejte do terminálu následující příkazy:

```
$ npm install #stažení balíčků z package.json
$ npm run build #kompilace z TypeScript do JavaScript
$ npm run dev #spuštění lokálního http serveru
```
